# Testing

- For the unit tests:
    ```bash
    make unittest
    ```
- Testing the examples from the documentation:
    ```bash
    make doctest
    ```
- Static type checker:
    ```bash
    make mypy
    ```

Or all these tests by simply removing the target:
```bash
make
```
